# Тест функциональности кнопки "Я получатель — хочу оформить доставку"

## Что должно произойти при клике на кнопку:

1. **Плавная прокрутка** к форме "откуда/куда" в hero секции
2. **Установка фокуса** на поле "Откуда" через 800мс после начала прокрутки
3. **Пользователь может сразу начать ввод** города отправления

## Реализованные изменения:

### CalculatePage.jsx
- Добавлен `fromCityInputRef` для управления фокусом
- Добавлен обработчик `handleRecipientDelivery()` 
- Обработчик находит первую форму на странице и плавно прокручивает к ней
- Через 800мс устанавливает фокус на поле "Откуда"
- Кнопка получила `onClick={handleRecipientDelivery}`

### CityForm.jsx  
- Добавлен проп `fromCityInputRef` (опциональный)
- Ref передается в CityInput для поля "Откуда"

### CityInput.jsx
- Добавлен проп `inputRef` (опциональный)  
- Компонент использует переданный ref или создает внутренний
- Все обращения к inputRef заменены на currentInputRef

## Порядок тестирования:
1. Открыть главную страницу `/calculate`
2. Прокрутить вниз к синей кнопке "Я получатель — хочу оформить доставку"
3. Кликнуть на кнопку
4. Ожидаемый результат: страница плавно прокручивается к форме и фокус устанавливается на поле "Откуда"
